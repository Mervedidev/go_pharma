Politique de confidentialité

Chez Go Pharma, la protection de vos données personnelles est une priorité. Cette politique de confidentialité vous explique comment nous collectons, utilisons et protégeons vos informations.

1. Données collectées

Lorsque vous utilisez notre site ou passez une commande, nous collectons certaines informations personnelles, telles que :

  • Votre nom
  • Votre numéro de téléphone
  • Votre adresse e-mail
  • Votre adresse de livraison
  • Le nom des médicaments commandés

2. Utilisation des données

Nous utilisons vos données uniquement dans le but de :

  • Traiter vos commandes
  • Vous contacter en cas de besoin
  • Améliorer nos services

3. Partage des données

Vos données ne sont jamais revendues. Elles sont uniquement partagées avec :

  • Nos livreurs (pour effectuer la livraison)
  • Nos partenaires techniques (hébergeurs, outils d’e-mailing, etc.)

4. Sécurité

Nous mettons en œuvre toutes les mesures nécessaires pour garantir la sécurité de vos données (certificats SSL, accès restreints, etc.).

5. Vos droits

Vous avez le droit de :

  • Demander une copie de vos données
  • Modifier ou corriger vos informations
  • Demander la suppression de vos données

Pour toute demande, écrivez-nous à contact@gopharma.com.

6. Cookies

Notre site utilise des cookies pour améliorer votre expérience. Vous pouvez les accepter ou les refuser via les paramètres de votre navigateur.

7. Modifications

Cette politique peut être modifiée à tout moment. La version la plus récente sera toujours disponible sur cette page.
